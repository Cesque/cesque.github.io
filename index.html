Hi, I'm Daniel James
I'm a recent Computer Science graduate from the University of Birmingham, now looking for a job in front-end development. I'm passionate about creativity and technology, as well as finding interesting and innovative ways to combine them.
I have experience with the big web technologies - HTML, CSS and Javascript. In particular, I enjoy learning about new and cutting edge technologies, as well as learning new frameworks and libraries. For example, I wrote a Rubik's Cube solve timer using vue.js supported by features such as CSS grid and ES6 module support.
Alongside the front-end work, I also program in Node.js, creating server-side applications and other projects. For example, the cube timer also contains an API which supplies random intial states for Rubik's Cubes.
Another project is qcpu, a fictional CPU implementation written in Node.js, with an included assembler from a specially created assembly language. Using Browserify I used the CPU emulator module in a web interface which lets users write and execute their own assembly code for the CPU.
